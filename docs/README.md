## ğŸ“ Glossary of Terms

### **Technical Terms**
- **Mono.co:** Nigerian open banking platform (Okra alternative)
- **Okra.ng:** Original open banking provider (currently down)
- **NIBSS:** Nigeria Inter-Bank Settlement System
- **NIP:** NIBSS Instant Payment (for transfers)
- **PCI DSS:** Payment Card Industry Data Security Standard
- **NDPR:** Nigeria Data Protection Regulation
- **AES-256:** Advanced Encryption Standard (encryption algorithm)

### **Banking Terms**
- **Account Aggregation:** Combining multiple bank accounts in one view
- **Open Banking:** Banks allowing third-party access via APIs
- **PSP:** Payment Service Provider
- **PSB:** Payment Service Bank (CBN license type)
- **KYC:** Know Your Customer (identity verification)
- **AML:** Anti-Money Laundering

### **Project Terms**
- **Direct Integration:** Connecting directly to bank APIs (no aggregator)
- **Hybrid Architecture:** Using both direct + aggregator approaches
- **Velocity Checks:** Fraud detection based on transaction frequency
